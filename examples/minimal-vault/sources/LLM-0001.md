# Attention Is All You Need

## One-Sentence Contribution

The paper introduced the Transformer, a sequence transduction architecture based
on multi-head self-attention that reached 27.3 BLEU on WMT 2014 English-German
with the base model and 28.4 BLEU with the big model.

## Core Idea

Recurrent and convolutional models process sequence structure indirectly through
time steps or local windows. The Transformer replaces that path with
self-attention, allowing each token to attend directly to other positions in the
sequence. Positional encodings inject order information that attention alone
does not provide.

## Interpretation

The important claim is not simply that attention works. Earlier models already
used attention. The durable contribution is showing that attention can become
the main sequence modeling mechanism when paired with residual layers, feed
forward blocks, normalization, and positional encodings.
