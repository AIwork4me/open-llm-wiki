# Attention Is All You Need

> Transformer：纯 attention 架构，抛弃 RNN/CNN。8 头注意力，6 层 encoder+decoder，在 WMT 英德翻译上达到 SOTA，同时训练速度快 10 倍。改变了整个 NLP 的方向。

## 核心思路

之前做 seq2seq 用 LSTM/GRU，序列计算是瓶颈——无法并行。Transformer 的洞察：**attention 本身就是够了**，不需要循环结构。用 self-attention 让每个位置直接看所有其他位置，完全并行。

关键创新：
1. **Multi-Head Attention** — 8 个头并行关注不同子空间
2. **Positional Encoding** — 用正弦/余弦注入位置信息（attention 本身不感知位置）
3. **Scaled Dot-Product** — 除以 √d_k 防止梯度消失

## 我的判断

Transformer 的核心贡献不是 attention 本身（之前就有），而是证明了**纯 attention 架构可以工作**。这个洞察看似简单，实际上打破了"序列数据必须用序列模型"的教条。之后整个 LLM 领域的爆发都建立在这个基础之上。

Positional Encoding 是个优雅的补丁——attention 不感知位置，所以手动注入。这个设计后来被 RoPE（旋转位置编码）和 ALiBi 取代，但思路一脉相承。
